# Quick Start Guide

## Get Running in 30 Seconds

### 1. The app is already set up! Just run:

```bash
cd /Users/sagardesai/Files/Documents/Personal/Repos/multivideo
npm run dev
```

### 2. Open your browser to:

```
http://localhost:3000
```

### 3. Start watching videos:

1. Click on button **1** (to select first quadrant)
2. Paste a YouTube URL: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
3. Click **Load**
4. Repeat for quadrants 2, 3, and 4 with different videos
